# dsdeco
Discord's decision to only keep certain non-Nitro users' decoration items on display in their profile for a limited time, thereby luring people into subscribing to Nitro to use them indefinitely, is really frustrating. 

So, in case you have a "courtesy" avatar decoration in use, run this script to retrieve the corresponding file, and make sure you keep it away from Discord's ambitious hands, no one knows... 

**USE THE OBTAINED FILE WISELY. EVEN IF YOU REALLY LIKE IT OR HAVE INTERACTED WITH THE DECORATION ONLY ONCE OR TWICE ON DISCORD, IT DOES _NOT_ BELONG TO ITS END USERS.**
